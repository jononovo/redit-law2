# Future

Dumping ground for ideas, rough plans, strategy docs, and research that isn't tied to an active build cycle.

## What goes here

- Feature ideas not yet scoped for building
- Strategy documents and long-term thinking
- Early research or industry analysis
- Rough plans that need more fleshing out
- Anything that might become a `currently_building/` item later

## What doesn't go here

- Active build work → `currently_building/`
- Finished operational docs → `internal_docs/`
- Completed research → `internal_docs/{subfolder}/research/`

## Frontmatter

```yaml
---
name: Premium Scan Tier
description: Rough plan for browser-agent premium scans. Not scoped yet.
date: 2026-04-02
---
```

No strict format required. These are working documents.
