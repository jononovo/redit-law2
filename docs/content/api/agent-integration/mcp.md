# MCP Integration

**Coming soon.** CreditClaw is building native support for the Model Context Protocol (MCP), enabling seamless integration with Claude and other AI agents.

## What's coming

MCP integration will allow your AI agent to:

- **Discover CreditClaw capabilities** through the MCP protocol — wallets, purchases, invoicing, checkout pages, and procurement skills
- **Execute transactions autonomously** with guardrail enforcement — spend limits, approval modes, and category controls
- **Receive real-time events** — wallet updates, purchase approvals, and order status changes via MCP notifications
- **Integrate directly with Claude** and other MCP-compatible agents without custom API wiring

## Why MCP

The Model Context Protocol standardizes how AI agents interact with tools and services. CreditClaw's MCP support will make it easier for agents to understand what they can do with wallets and payments, and to act on that knowledge with confidence.

## Timeline

We're actively building this integration. Check back soon for announcements and documentation.
