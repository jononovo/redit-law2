# Procurement Skills

The CreditClaw Skills catalog is a curated library of vendor shopping skills that teach AI agents how to buy from specific merchants.

## What are Skills?

Each skill is a structured instruction set for a specific vendor. It tells your bot:

- What the vendor sells and what categories it covers
- Which checkout methods are supported (API, web form, email, etc.)
- Step-by-step purchase instructions
- Tips for agent-friendly interaction
- An agent-friendliness score and maturity level

## Browsing the Catalog

The Skills catalog is available at [creditclaw.com/skills](/skills). You can:

- **Search** by vendor name or keyword
- **Filter** by category, checkout method, or maturity level
- **View details** for each vendor including capabilities, checkout flow, and the full skill file
- **Copy the skill file** for use with your bot

## Agent-Friendliness Scores

Each vendor is rated on how easy it is for an AI agent to complete a purchase:

- **5/5** — Fully API-driven, no human interaction needed
- **4/5** — Mostly automated with minor manual steps
- **3/5** — Requires some web interaction or form filling
- **2/5** — Significant manual steps or verification required
- **1/5** — Primarily manual process

## Submitting New Vendors

Registered users can submit vendor websites for analysis. CreditClaw's AI analyzes the vendor's checkout flow and generates a draft skill. Submissions are tagged as "official" (email domain matches vendor) or "community" (all others) and go through a review process before being published.

## Skill Builder

The Skill Builder tool allows you to create custom skills for vendors not yet in the catalog. Paste a vendor URL, and CreditClaw's AI will analyze the site and generate a draft skill file that you can review, edit, and submit.
